# 🤖 Project 1: Rule-Based AI Chatbot
### DecodeLab Internship

---

## 📌 About This Project
This is a simple rule-based chatbot built in Python.
It responds to user messages using if-else logic and runs in a continuous loop.
No machine learning — just pure Python logic.

---

## 🎯 What It Does
- Greets the user (hi, hello, hey)
- Tells jokes and answers questions
- Shows current time and date
- Solves math using words (add 5 and 3) or symbols (10 + 5)
- Exits when user types bye or exit

---

## 🛠️ Skills Used
- Python if-else logic
- String handling
- Loops (while loop)
- Basic AI chatbot concepts
- Regular expressions (re module)
- Date and time handling (datetime module)

---

## 💬 Example Chat
User: hello
Bot:  Hello! 

User: tell me a joke
Bot:  Why do programmers prefer dark mode? Because light attracts bugs!

User: add 10 and 5
Bot: The answer is 15.0

User: bye
Bot:  Goodbye! Have a nice day!
